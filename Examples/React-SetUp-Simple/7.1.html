Example 1

While a react component describes what to do, nothing will be rendered unless we tell react so. This is done through ReactDOM.render. As you notice we are writing HTML withing JavaScript. This is possible because we are using the type="text/babel" instead of the type="text/javascript". This way or writing is a variation on javascript that its called JSX. Attetion; we can write more than just plain HTML!
