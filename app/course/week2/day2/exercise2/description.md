#### Find All lis

**Snippet:**

```html
<div id = "exerciseTest">
    <ul>
        <li name="color"  value="Red">Sun</li>

        <li name="accept"  value="Evil Plans">Hill</li>
        <span>  Dale</span>
    </ul>
        <button id="button1">Click to see the effect</button>

```
