### Chop string

Write a function called ```stringChop``` to chop a string into chunks of a given length
Example:

```jsx
stringChop('w3resource'); //#=> ["w3resource"] 
stringChop('w3resource',2);  //#=>  ["w3", "re", "so", "ur", "ce"]
stringChop('w3resource',3); //#=>  ["w3r", "eso", "urc", "e"]
```
